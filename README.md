# MoveEngine 

Strategic Tic Tac Toe built with React featuring:

- Player vs Player mode
- Player vs AI mode (rule-based heuristic engine)
- Editable player names
- Clean component architecture
- Game state management with React hooks

---

##  Features

- Toggle between PVP and AI mode
- Rule-based AI (win → block → center → corner → side)
- Smooth UI interactions
- Restart functionality
- Responsive design
- Clean separation of components

---

## AI Logic

The AI is implemented using a rule-based heuristic approach:

1. Win if possible
2. Block opponent win
3. Take center
4. Take opposite corner
5. Take any corner
6. Take side

This ensures competitive gameplay without using minimax.
